# DBZ Scouter User Guide

This project is designed to be a working screen module for a Dragon Ball scouter cosplay.

![DBZ Scouter banner](https://github.com/2omethingBaD/DBZ-Scouter/blob/main/img/20260715_165016.jpg?raw=true)

## Assembly

For easy assembly simply plug the included lithium polymer battery into the JST connector on the PCB. be sure the red wire matches up with the small + on the PCB when plugging it in, as shown in the images below.
