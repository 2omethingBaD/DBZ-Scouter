# DBZ Scouter User Guide

This project is designed to be a working screen module for a Dragon Ball scouter cosplay.

![DBZ Scouter banner](https://github.com/2omethingBaD/DBZ-Scouter/blob/main/img/20260715_165016.jpg?raw=true)

---

## Assembly

- For easy assembly simply plug the included lithium polymer battery into the JST connector on the PCB. be sure the red wire matches up with the small + on the PCB when plugging it in, as shown in the images below.

![DBZ Scouter JST plugged in](https://github.com/2omethingBaD/DBZ-Scouter/blob/main/img/20260715_164435.jpg?raw=true)
![DBZ Scouter JST plug](https://github.com/2omethingBaD/DBZ-Scouter/blob/main/img/20260715_165133.jpg?raw=true)

---

## Usage

- Once turned on, and after the splash screen the sytem will play the first "scanning" animation befor settling on the end screen for a few seconds then entering a sleep mode with a plain black screen until the button 
is pressed and held down for a minimum of 2 seconds

- the button can also be used to reset the animation from the "scanning" screen at any point during the animation.

- a button cap extendor was also included as an optional choice to extend the button if needed

![DBZ Scouter input button](https://github.com/2omethingBaD/DBZ-Scouter/blob/main/img/20260715_165102.jpg?raw=true)
![DBZ Scouter button cap extendor](https://github.com/2omethingBaD/DBZ-Scouter/blob/main/img/20260715_165237.jpg?raw=true)

---

## Charging

- The included lithium polymer battery can be easily charged via the USBC port on the MCU while the battery is attached to the PCB.

- the red led on the MCU will flash as the battery is charging and will stop flashing and turn off completely once the battery is fully charged.

![DBZ Scouter charging](https://github.com/2omethingBaD/DBZ-Scouter/blob/main/img/20260715_165617.jpg?raw=true)
